## 工作流程

### Step 1: 撰写招生文案

根据目标受众（高中毕业生/家长），撰写短视频文案。文案风格要：
- **开头抓人**：3秒内吸引注意力
- **痛点切入**：高考压力大、分数线高、学费贵等
- **方案展示**：哈萨克国立大学作为替代选择
- **数据支撑**：QS排名、学费、专业等具体数字
- **结尾引导**：评论区咨询、私信获取更多信息

#### 文案模板

```
🔥【开头】高考不是唯一出路！

💡【痛点】国内985分数线越来越高，学费动辄5万+，竞争激烈到窒息...

✨【方案】但你知道吗？QS世界排名150+的哈萨克国立大学，学费只要1-3万/年！

📚【亮点】
- 14个学院，300+专业，石油工程、国际关系、计算机...
- 英语授课项目，不用学俄语也能读
- 一带一路核心国家，毕业就业前景广阔
- 阿拉木图——中亚最美城市，生活成本低

🎓【引导】想了解更多？评论区扣1，私信发你完整招生简章！
```

#### 不同风格文案

**风格A - 激励型**（适合高中毕业生）：
```
别让高考定义你的人生！
QS150+名校，1-3万/年学费
```

**风格B - 算账型**（适合家长）：

**风格C - 故事型**（适合软性推广）：

### Step 2: 通过豆包生成视频

使用 `com.larus.nova`（豆包App）生成视频。

> ⚠️ **注意**：豆包 ≠ 即梦AI（即梦包名是 `com.bytedance.dreamina`）。豆包内置了视频生成功能（视频2.0），无需使用即梦。

#### 豆包App界面结构

底部5个Tab：**对话** | **智能体** | **创作** | **通知** | **我的**

创作Tab内的分类：**发现** | **视频2.0** | **P图** | **萌宠** | **宝宝**

#### 操作流程

1. **启动豆包App**：
```bash
launch_app(package="com.larus.nova")
```

2. **进入创作Tab**：
   - 点击底部第3个Tab「创作」（坐标约 x:633, y:2684）

3. **进入视频2.0**：
   - 点击顶部「视频2.0」分类标签（坐标约 x:420, y:383）

4. **开始创作**：
   - 点击「开始创作」按钮（坐标约 x:672, y:2388）

5. **切换到视频模式**：
   - 在创作界面，点击「视频」标签（坐标约 x:388, y:853）
   - 图片模式默认模型4.5，视频模式默认模型2.0Fast

6. **输入视频描述**：
   - 点击输入框「描述你想要创作的视频.」（坐标约 x:376, y:1590）
   - ✅ **使用 `type_text` 工具输入中文**（已验证可用，通过 ACTION_SET_TEXT 方式注入）
   - 示例：`type_text(text="大学校园宣传片风格，展示哈萨克国立大学的现代化教学楼、图书馆、实验室和学生活动场景，配合阿拉木图城市风光和雪山背景，色调温暖明亮，节奏适中，适合招生宣传，时长15-30秒，竖屏9:16")`
   - ⚠️ 如果 `type_text` 失败，替代方案：
     - 方案A：使用语音输入功能（点击右侧「の」按钮，然后「按住说话」）
     - 方案B：使用英文描述（简化版）
     - 方案C：通过豆包对话界面，发送文字消息请求生成视频
   - 也可点击右侧「..」按钮展开更多输入方式

7. **配置选项**：
   - **参考图**：可上传参考图片（坐标约 x:216, y:1180）
   - **创建分身**：创建数字人分身（坐标约 x:276, y:1404）
   - **模型选择**：视频模式默认「2.0Fast」，点击可切换（坐标约 x:640, y:1404）
   - **玩法**：选择视频风格模板（坐标约 x:968, y:1406）
   - **比例**：选择视频比例 1:1 / 9:16 / 16:9（坐标约 x:1204, y:1406）

8. **生成视频**：
   - 输入描述后，点击发送/生成按钮
   - 等待视频生成（通常1-5分钟）

9. **发抖音**（主路径，最快捷）：
   - 视频生成完成后，在视频结果页面，点击右上角**「分享」**按钮
   - 弹出分享面板，底部有5个选项：**存为视频**、**发抖音**、**微信好友**、**朋友圈**、**QQ好友**
   - 点击**「发抖音」**按钮（坐标约 x:416, y:2643）
   - 等待加载完成（视频数据传输到抖音，约3-10秒）
   - 自动跳转到**抖音视频编辑页面**

> 💡 **推荐路径**：豆包「发抖音」→ 抖音编辑页 → 下一步 → 发布页，一步到位，无需中间保存。
> ⚠️ **重要**：抖音编辑页底部有「朋友日常」和「下一步」两个按钮。**「朋友日常」是直接发布的快捷操作**（仅朋友可见），**不要点它**！要点**「下一步」**进入发布设置页，在那里可以选择公开/私密、填写文案。

10. **保存视频**（备用方案）：
   - 如果分享失败或需要保留视频文件，可点击「保存」将视频保存到本地相册
   - 然后手动打开抖音 → 点击「+」→ 相册 → 选择视频 → 发布

#### 视频提示词建议

```
大学校园宣传片风格，展示哈萨克国立大学的建筑、图书馆、实验室、学生活动，
配合阿拉木图城市风光和雪山背景，色调温暖明亮，节奏适中，
适合招生宣传，时长15-30秒
```

#### 「做同款」功能

在视频2.0发现页，可以浏览他人作品，点击「做同款」按钮，基于他人作品的创作信息修改生成自己的视频。适合快速复用优质模板。

### Step 3: 发布到抖音

#### 主路径：从豆包分享跳转（推荐）

视频生成完成后，在豆包的视频结果页面：

1. **点击右上角「分享」按钮**
2. **在分享面板中选择「抖音」**
3. **等待加载完成**（约3-10秒，视频数据从豆包传输到抖音）
4. **自动跳转到抖音发布页面**，此时视频已加载好

在抖音发布页面：

5. **填写文案和标签**：
   - 点击描述输入框
   - 使用 `type_text` 输入招生文案，例如：
   ```
   type_text(text="高考不是唯一出路！QS150+名校哈萨克国立大学，年费1-3万，留学新选择 #留学 #哈萨克斯坦")
   ```

6. **选择发布类型**：
   - ⚠️ **注意**：「朋友日常」是仅朋友可见的，不是公开发布！
   - 如需公开发布，应切换为「公开」类型

7. **点击「发布」**

#### 备用路径：从抖音相册选择

如果豆包分享失败，可先保存视频到相册，再从抖音发布：

1. **在豆包保存视频**：点击「保存」按钮，视频保存到相册
2. **启动抖音App**：
```bash
launch_app(package="com.ss.android.ugc.aweme")
```
3. **点击「+」号**：底部中间的拍摄/发布按钮
4. **点击「相册」**：在拍摄页面底部选择「相册」
5. **筛选视频**：点击顶部「视频」标签，找到豆包生成的视频
6. **选择视频** → **点击「下一步」** → **编辑（可选）** → **填写文案** → **发布**

## 注意事项

1. **文案合规**：避免夸大宣传，数据要准确，不违规承诺
2. **视频时长**：抖音推荐15-30秒短视频，不超过60秒
3. **视频比例**：抖音竖屏推荐9:16，横屏推荐16:9
4. **发布时间**：建议在晚上7-10点发布，流量高峰期
5. **互动引导**：文案中要有明确的互动引导（评论/私信）
6. **系列内容**：可以制作系列视频（专业介绍、城市风光、学生故事等）
7. **豆包等待**：视频生成可能需要1-5分钟，需耐心等待
8. **抖音审核**：新账号发布可能需要审核，注意内容合规
9. **豆包≠即梦**：豆包(`com.larus.nova`)和即梦AI(`com.bytedance.dreamina`)是两个不同的App，视频生成使用豆包内置的「视频2.0」功能
10. **分享→抖音是主路径**：豆包生成视频后，直接点右上角「分享」→选「抖音」，比保存相册再选更快捷
11. **发布类型**：抖音发布时注意选择「公开」而非「朋友日常」（朋友日常仅朋友可见）
12. **分享加载**：豆包分享到抖音时需等待加载完成（3-10秒），不要提前操作

## 已知技术限制（2025-01-22更新）

### 中文文本输入 — ✅ 已解决

**问题描述**：之前 `type_text` 等工具无法输入中文文本。

**解决方案**：`type_text` 工具通过 ACTION_SET_TEXT 方式注入文本，**已验证可以成功输入中文**。

**使用方法**：
```python
type_text(text="你的中文内容")
```

**注意事项**：
- 输入前确保目标输入框已获得焦点（先点击输入框）
- 部分App可能有输入拦截，如果失败可尝试先点击输入框再输入
- 如果 `type_text` 失败，可尝试以下替代方案：
  - 方案A：使用语音输入功能（点击右侧「の」按钮，然后「按住说话」）
  - 方案B：使用英文描述（简化版）
  - 方案C：通过豆包对话界面，发送文字消息请求生成视频
